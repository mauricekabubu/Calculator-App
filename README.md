# Calculator App

## Overview
This is a **simple Calculator application** built using **Python** and the **Tkinter** library for the graphical user interface (GUI).  
The app performs basic arithmetic operations and was created as a **beginner Python project** to practice GUI development and event handling.

---

## Features

- Basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Interactive **buttons** for numbers and operations  
- **Entry field** to display input and results  
- Simple and easy-to-use **Tkinter GUI**

---

## Tech Stack

- **Python** – Programming language  
- **Tkinter** – GUI library for Python  

---

## Goal of the Project

The main goals of this project were:

1. Learn how to build a **GUI application** using Tkinter  
2. Understand **buttons, entry widgets, and layouts**  
3. Practice handling **user input and events**  
4. Strengthen **Python basics** through a practical project  

---

## What I Learned

During this project, I learned:

- How to create **windows and widgets** using Tkinter  
- How to use **buttons and entry fields**  
- How to handle **button click events**  
- How to structure a **simple Python GUI application**  
- Basic debugging and UI logic
